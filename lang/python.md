### String Manipulation

```python
>>> s1 = "this is a string"
>>> s1
'this is a string'
>>> s1 = 'Also a string'
>>> s1
'Also a string'
>>> s1 = r"a\raw\string\in\double\quotes"
>>> s1
'a\\raw\\string\\in\\double\\quotes'
>>> s1 = """A
... multiline
... string"""
>>> s1
'A\nmultiline\nstring'
>>> " ".join(["List", "or", "Tuple"])
'List or Tuple'
>>> "A formatted {}".format("string")
'A formatted string'
>>> "Square root of {} is equal to {:.2f}".format(3, math.sqrt(3))
'Square root of 3 is equal to 1.73'
```
